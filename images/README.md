## Image Guidelines

- Use high-quality images that showcase your best work
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Keep file sizes reasonable (< 500KB for portfolio, < 1MB for hero images)
- Use consistent aspect ratios within each category
- Ensure images are properly licensed and you have permission to use them

## Notes

If images are not added, the website will automatically display placeholder images.
You can add images gradually and the website will show placeholders for missing images.
